# T12 — Build review queue model

## Summary
Create a shared review queue for weak or due items.

## Goals
- Support repetition and weak-item practice
- Prepare for spaced-review behavior

## Scope
- Review item model
- Add/remove/update queue
- Mark due/completed
- Integrate with quiz mistakes and saved items

## Deliverables
- Review queue service
- Review item type

## Acceptance Criteria
- Mistakes can be added to review
- Due items can be listed
- Completed review items update state correctly
