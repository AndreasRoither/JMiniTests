# T19 — Build quiz components

## Summary
Create shared interactive quiz primitives.

## Goals
- Avoid plugin-specific quiz duplication
- Support multiple study modes later

## Scope
- Multiple choice
- Typed answer
- Ordering
- Matching
- Feedback states
- Correct/incorrect handling

## Deliverables
- Shared quiz component library

## Acceptance Criteria
- Multiple plugins can reuse the same components
- Immediate feedback works
- Quiz components are keyboard and touch friendly
