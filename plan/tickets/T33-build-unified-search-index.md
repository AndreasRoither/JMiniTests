# T33 — Build unified search index

## Summary
Create a unified search index for lessons, plugins, exercises, grammar, vocab, and tests.

## Goals
- Make discovery fast and consistent
- Allow search across content and interactive items

## Scope
- Searchable item model
- Content indexing
- Plugin indexing
- Query API
- Result grouping

## Deliverables
- Search index service
- Search query helpers

## Acceptance Criteria
- User can search across multiple content kinds
- Results include levels/topics/tags
- Search works with saved and in-progress items where appropriate
