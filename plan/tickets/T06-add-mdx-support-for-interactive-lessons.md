# T06 — Add MDX support for interactive lessons

## Summary
Support embedded interactive widgets in selected lessons.

## Goals
- Keep most lessons simple Markdown
- Allow React widgets in lessons when needed

## Scope
- Install MDX support
- Register safe embedded components
- Define authoring rules for MDX usage
- Ensure widget hydration is selective

## Deliverables
- MDX lesson support
- Embedded widget examples

## Acceptance Criteria
- MDX lessons render correctly
- React widgets can appear in MDX
- Full plugin apps are not required for small inline widgets
