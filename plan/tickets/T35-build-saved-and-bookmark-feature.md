# T35 — Build saved and bookmark feature

## Summary
Allow users to save and bookmark lessons, vocab, grammar, plugins, and tests.

## Goals
- Support revisit and personal curation
- Improve self-directed study

## Scope
- Save/bookmark controls
- Saved items page
- Saved filter integration
- Persistence hooks

## Deliverables
- Saved/bookmark feature
- Saved items page

## Acceptance Criteria
- Multiple item types can be saved
- Saved items persist across reload
- Saved content is filterable and searchable
