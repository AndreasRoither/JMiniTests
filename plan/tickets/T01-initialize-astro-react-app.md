# T01 — Initialize Astro + React app

## Summary
Create the base application using Astro with React integration and TypeScript.

## Goals
- Create the initial project structure
- Add Astro + React support
- Add TypeScript and Tailwind
- Set up scripts for linting, formatting, and testing

## Scope
- Create Astro project
- Install React integration
- Install Tailwind
- Configure tsconfig and aliases
- Add package scripts

## Deliverables
- Running Astro app
- React components rendering in Astro pages
- Tailwind working
- Basic dev/build/test scripts

## Acceptance Criteria
- `npm run dev` starts the app
- React component renders inside an Astro page
- Tailwind styles apply correctly
- TypeScript path aliases work
