# T22 — Build plugin analytics summary

## Summary
Create summary views for plugin usage and performance.

## Goals
- Show where time is spent
- Highlight weak areas

## Scope
- Study time by plugin
- Completion counts
- Mistake-heavy plugins/topics
- Recent activity

## Deliverables
- Analytics summary components

## Acceptance Criteria
- Usage summaries read from tracked events
- Weak areas are visible
- Summaries degrade safely if little data exists
