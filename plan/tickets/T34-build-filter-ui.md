# T34 — Build filter UI

## Summary
Build reusable filtering UI for browse and search pages.

## Goals
- Help users narrow content quickly
- Support level/topic/status-driven study

## Scope
- Level filter
- Topic filter
- Type filter
- Status filter
- Saved/bookmarked filter
- Due-for-review filter

## Deliverables
- Reusable filter components

## Acceptance Criteria
- Filters work alone and in combination
- Filter state is reflected in results
- UI remains usable on mobile
