# T04 — Configure content collections

## Summary
Set up structured content collections for lessons, grammar, vocabulary, exercises, and tests.

## Goals
- Create a typed content pipeline
- Support future content growth

## Scope
- Define content schemas
- Add collections for lessons, vocab, grammar, exercises, tests
- Add validation helpers
- Define slug and ID conventions

## Deliverables
- Typed collections
- Validated frontmatter

## Acceptance Criteria
- Invalid content fails validation
- Content can be queried by collection
- Levels, topics, and tags are supported in schemas
