# T38 — Build first mock test flow

## Summary
Create the first end-to-end mock test experience.

## Goals
- Prove the assessment model with a real flow
- Support level/topic-driven testing

## Scope
- Choose level
- Choose topic or mixed set
- Start test
- Submit answers
- Show result summary
- Save result and mistakes

## Deliverables
- First mock test flow

## Acceptance Criteria
- User can complete a mock test
- Results are shown at the end
- Result data is saved for later review
