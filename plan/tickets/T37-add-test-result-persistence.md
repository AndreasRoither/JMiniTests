# T37 — Add test result persistence

## Summary
Persist test history and useful score summaries.

## Goals
- Track assessment progress over time
- Support review from mistakes

## Scope
- Test attempt model
- Best score and latest score
- Mistake capture
- Level/topic result grouping

## Deliverables
- Test persistence model
- Test result storage

## Acceptance Criteria
- Finished tests create stored attempt records
- Best and latest scores are retrievable
- Mistakes can feed review items
