# T05 — Build Markdown lesson renderer

## Summary
Create the lesson page rendering flow for Markdown content.

## Goals
- Render lessons cleanly
- Show learning metadata
- Support reading progression

## Scope
- Lesson page template
- Title, metadata, tags, level display
- Next/previous lesson navigation
- Estimated time display
- Completion state integration

## Deliverables
- Reusable lesson page
- Metadata-aware lesson UI

## Acceptance Criteria
- Markdown lessons render correctly
- Level/topic/tags display correctly
- Navigation between lessons works
