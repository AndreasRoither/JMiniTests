# T36 — Add assessment mode framework

## Summary
Create shared support for learn, practice, review, and test modes.

## Goals
- Standardize how plugins and tests run
- Prepare for timed and scored assessments

## Scope
- Mode definitions
- Shared scoring rules
- Mode-aware UI states
- Plugin capability integration

## Deliverables
- Assessment mode framework

## Acceptance Criteria
- Plugins can declare supported modes
- UI adapts between practice and test contexts
- Core scoring model exists
