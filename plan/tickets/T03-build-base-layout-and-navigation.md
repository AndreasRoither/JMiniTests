# T03 — Build base layout and navigation

## Summary
Create the application shell and main navigation.

## Goals
- Build responsive layout for phone, tablet, and desktop
- Add main navigation for core study areas

## Scope
- Header and primary navigation
- Footer or compact mobile navigation
- Base page container
- Home page
- Sections for lessons, practice, tests, review, saved, dashboard

## Deliverables
- Working app shell
- Responsive navigation

## Acceptance Criteria
- Navigation works across device sizes
- Core pages are reachable
- Layout does not break on small screens
