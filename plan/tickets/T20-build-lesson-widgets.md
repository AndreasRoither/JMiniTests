# T20 — Build lesson widgets

## Summary
Create small reusable widgets for lesson pages and summaries.

## Goals
- Standardize lesson metadata and progress UI
- Improve continuity across content pages

## Scope
- Progress bar
- Completion badge
- Estimated time widget
- Due review indicator
- Saved/bookmark control

## Deliverables
- Shared lesson widgets

## Acceptance Criteria
- Widgets can be placed on lesson pages
- Saved/completed states update correctly
- Widgets are responsive
