# T31 — Add JLPT level taxonomy

## Summary
Add N5–N1 support across content, plugins, and progress.

## Goals
- Make level-based learning a core feature
- Support future filtering and testing

## Scope
- Level enums
- Multi-level content support
- Level metadata in content and plugin schemas
- Level-aware dashboard hooks

## Deliverables
- Shared level taxonomy

## Acceptance Criteria
- Content can declare one or more levels
- Plugins include levels
- Progress can be grouped by level
