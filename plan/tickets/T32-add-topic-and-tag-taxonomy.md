# T32 — Add topic and tag taxonomy

## Summary
Define a consistent topic and tagging system.

## Goals
- Support search, filters, and cleaner organization
- Reduce inconsistent labels

## Scope
- Topic list
- Tag guidance
- Shared metadata helpers
- Validation where reasonable

## Deliverables
- Topic/tag taxonomy rules

## Acceptance Criteria
- Core content types use consistent topics
- Search/filter logic can rely on tags/topics
- Duplicate near-synonym tags are reduced
